# Açai zoo
Açaí Zoo 🦁 - Uma experiência lúdica e personalizada de açaíteria temática com confeitaria criativa.
## 🚀 Sobre o Projeto
O diferencial do **Açaí Zoo** é a união do açaí artesanal com a confeitaria criativa. Cada tigela é finalizada com moldes artesanais de chocolate em formato de animais, criando um produto visualmente atraente e exclusivo.

## 🎨 Design & UX/UI
O projeto foca em uma experiência lúdica para o usuário:
* **Identidade Visual:** Cores vibrantes (Roxo, Verde e Marrom).
* **Interface:** Cardápio organizado por "espécies".
* **Responsividade:** Feito para funcionar em qualquer tela.
