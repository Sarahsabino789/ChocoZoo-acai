# ChocoZoo Açai 
ChocoZoo Açai 🦁 - Uma experiência lúdica e personalizada de açaíteria temática com confeitaria criativa.
## 🚀 Sobre o Projeto
O diferencial do **ChocoZoo Açaí** é a união do açaí artesanal com a confeitaria criativa. Cada tigela é finalizada com moldes artesanais de chocolate em formato de animais, criando um produto visualmente atraente e exclusivo.

## 🎨 Design & UX/UI
O projeto foca em uma experiência lúdica para o usuário:
* **Identidade Visual:** Identidade visual temática e orgânica, utilizando uma paleta baseada no produto (roxo, verde e marrom) para criar um design imersivo e de alto contraste).
* **Interface:** Cardápio organizado por "espécies".
* **Responsividade:** Feito para funcionar em qualquer tela.

## 🛠️ Tecnologias

* **CSS3**
